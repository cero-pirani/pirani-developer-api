# PIRANI documentation repository


Para editar el contenido de la página https://developer.pirani.co/

Deben descargar el repo https://github.com/cero-pirani/pirani-developer-api el cual tiene 1 solo archivo YML, ese archivo es el contenido que se ve en la página.

Para editarlo pueden crear una cuenta gratuita en https://redocly.com/redoc, subir el archivo e ir viendo los cambios en su editor, al terminar descargan o copian el contenido del archivo, lo remplazan en el repo y suben el commit.

Ese repo está contenado a una cuenta de https://redocly.com/redoc, que automáticamente leerá el YML y parseará el contenido para mostrarlo en la página https://developer.pirani.co/.
